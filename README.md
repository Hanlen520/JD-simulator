# JD-simulator
基于Python 3 京东商城模拟工具
## 主要功能
模拟京东商城登录请求、查询商品信息、提交订单等服务
## 目前已实现功能
### 登录功能
>   验证码扫码登录 </br>
>   密码登录 </br>
>   保存cookies </br>
> ※验证cookies是否过期 </br>
>   查看商品库存 </br>
>   查看商品价格 </br>
>   代码模块化 </br>
>   添加商品进入购物车 </br>
>   显示购物车信息 </br>
